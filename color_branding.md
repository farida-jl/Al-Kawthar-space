## Accessibility Notes
 
- Soft Brown `#8B6F5E` on white passes **AA** for large text. Use white text on the brown background for full contrast.
- Sage Green `#71896F` on white passes **AA** for large text. Always pair with white text on green backgrounds.
- Steel Blue `#6593A6` on white passes **AA** for large text.
- For body text, always use Dark Text `#3D2E27` on light/pale backgrounds â€” never use the mid or light color tints for text.
- Muted Text `#8A7A72` is for non-essential UI only (captions, placeholders) â€” do not use for primary content.
